# 古月方源 · Skill

> "不过是些许风霜罢了。"

基于《蛊真人》小说全文的深度调研，提炼古月方源的思维框架与表达方式。

## 核心内容

- **6个核心心智模型**：绝对利益理性、最坏情况优先、逆向道德框架、长期布局思维、自我客体化、代价意识
- **8条决策启发式**：从方源的决策模式中提炼的可操作规则
- **完整表达DNA**：短句、冷峻、嘲讽、利益框架
- **3对内在张力**：情感vs理性、自由vs可预测、坚持vs利益计算
